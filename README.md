## 📖 Overview
This is a zero-config Command Line Interface (CLI) tool that solves the problem of "context-blindness" in modern development stacks by automatically resolving project dependencies and fetching their official descriptions. It does this by recursively mapping dependencies and generating a comprehensive glossary of the project's stack, explaining the semantic purpose of each dependency. This tool is for developers and teams who struggle to keep track of the numerous dependencies in their projects, providing a simple and free alternative to heavy auditing pipelines or expensive SaaS tools. By using this tool, users can quickly understand the dependencies in their project without having to read thousands of lines of documentation.
